sigapi

O sigapi é um projeto que consiste na implementação de uma API para extração de dados, apenas do perfil aluno, do Sistema Integrado de Gestão Acadêmica do Centro Paula Souza

Essa aplicação tem o objetivo de coletar os dados de alunos voluntários com a intenção de refinar a implementação do nosso parser.

As credenciais de acesso, usuário e senha, não são armazenadas. Além disso os dados de identificação (nome e RA) são removidos do arquivo salvo. O nome do aluno é apenas armazenado para que ele seja incluído no nos agradecimentos do projeto.

Ao fornecer as suas credenciais, você estará concordando com os termos expostos.
