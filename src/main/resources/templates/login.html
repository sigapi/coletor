sigapi

Nosso projeto consiste na implementação de uma API para extração de dados, apenas do perfil aluno, do Sistema Integrado de Gestão Acadêmica do Centro Paula Souza

Essa aplicação tem o objetivo de coletar os dados de alunos voluntários com a intenção de refinar a implementação do nosso parser.

As credenciais de acesso (usuário e senha) não são armazenadas. Além disso, os dados de identificação (nome, RA e e-mail) são removidos do arquivo salvo. O nome do aluno é apenas armazenado para que ele seja incluído nos agradecimentos do projeto.

Ao fornecer as suas credenciais, você estará concordando com os termos expostos.
